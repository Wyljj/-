## 系统功能
### 1. 模板中心
- 功能 ：左侧显示可用的计算模板
- 当前模板 ：「监测断面计算」模板
- 模板描述 ：废气监测，包括流速相对方差 σr 和矩形排气筒/烟道当量直径 D 的计算
- 预留扩展 ：可添加含湿量、标干流量、含氧量折算、颗粒物浓度等计算模块
### 2. 数据录入
- 基本信息 ：
  - 样品类别、采样点位/断面编号、采样日期、检测人员、审核人员、备注
  - 矩形烟道长度 L (m)、矩形烟道宽度 W (m)、当量直径 D (m)
  - 当量直径自动计算：D = 2×L×W/(L+W)
- 测点流速录入 ：
  - 默认10个测点，支持动态添加和删除（至少保留2个）
  - 输入框始终显示边框，提高用户体验
  - 支持批量录入，可从文本粘贴数据
  - 实时计算偏差和偏差平方
### 3. 统计结果
- 核心计算 ：
  - 平均流速 v̄：所有测点流速的平均值
  - 样本标准差 s：衡量流速数据的离散程度
  - 流速相对均方差 σr：使用公式 σr = √[Σ(vᵢ−v̄)²/((n−1)×v̄²)]
  - 当量直径 D：根据矩形烟道长度和宽度计算
- 复核验证 ：
  - 偏差平方和 Σ(vᵢ−v̄)²
  - 复核计算 σr（公式与标准公式一致）
  - 差值分析：验证两种方法计算结果的一致性
- 结果判定 ：
  - 根据设定的限值自动判定结果是否合格
  - 显示判定结果和依据
- 操作按钮 ：
  - 保存记录：将计算结果保存到本地存储
  - 打印报告：打印计算结果和相关信息
  - 导出 CSV：将数据导出为CSV格式
  - 导出 JSON：将数据导出为JSON格式
### 4. 数据可视化
- 流速分布 ：柱状图显示各测点流速和平均流速
- 偏差分布 ：柱状图显示各测点流速与平均值的偏差
- 历史趋势 ：折线图显示历史计算结果和限值对比
### 5. 保存查询
- 历史记录 ：显示所有保存的计算记录
- 搜索功能 ：支持按点位、日期、结果、编号搜索
- 操作功能 ：加载历史记录、删除历史记录
### 6. 系统设置
- 默认参数 ：
  - 判定限值：可调整σr的判定标准
  - 小数位数：可调整计算结果的小数位数
- 系统操作 ：
  - 清除历史记录
  - 恢复默认设置
- 公式说明 ：显示流速相对均方差和当量直径的计算公式
## 技术实现
- 前端框架 ：原生HTML/CSS/JavaScript
- 图表库 ：Chart.js 4.4.0
- 存储 ：localStorage
- 布局 ：响应式设计，现代化的卡片式布局
- 交互体验 ：
  - 实时计算：输入数据时自动计算结果
  - 动态添加/删除测点
  - 批量数据导入
  - 数据可视化
  - 历史记录管理
  - 数据导出
## 系统优势
1. 功能完整 ：涵盖了废气监测中流速和当量直径的计算需求
2. 用户友好 ：界面美观，操作直观，支持批量录入和数据导出
3. 计算准确 ：公式正确，复核验证确保结果可靠性
4. 可扩展性 ：预留了模板扩展位置，可添加其他计算模块
5. 数据持久化 ：使用localStorage保存历史记录，方便后续查询和分析
6. 响应式设计 ：适配不同屏幕尺寸，提供良好的用户体验
## 系统使用流程
1. 输入基本信息 ：填写样品类别、采样点位、采样日期等信息
2. 输入烟道尺寸 ：填写矩形烟道长度L和宽度W，系统自动计算当量直径D
3. 输入流速数据 ：在测点流速录入表格中输入各测点的流速数据
4. 查看计算结果 ：切换到"统计结果"页签查看计算结果和判定
5. 保存记录 ：点击"保存记录"按钮保存当前计算结果
6. 查看历史记录 ：切换到"保存查询"页签查看和管理历史记录
7. 导出数据 ：在"统计结果"页签中点击"导出CSV"或"导出JSON"按钮导出数据
